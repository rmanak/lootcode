Split `s` into two non-empty parts `p + q`. The split is *good* when `p` and `q`
contain the **same number of distinct letters**. **Return how many good splits
exist.**

**Examples**
```
s = "aacaba"  ->  2
s = "abcd"    ->  1
s = "aaaaa"   ->  4
```

**Constraints:** `1 <= len(s) <= 10^5`, lowercase letters.
