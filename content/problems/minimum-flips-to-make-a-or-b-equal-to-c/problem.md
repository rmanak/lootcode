Given positive integers `a`, `b`, `c`, **return the minimum number of bit flips**
(each flip toggles a single bit of `a` or `b`) needed so that `a OR b == c`.

**Examples**
```
a = 2, b = 6, c = 5  ->  3
a = 4, b = 2, c = 7  ->  1
a = 1, b = 2, c = 3  ->  0
```

**Constraints:** `1 <= a, b, c <= 10^9`.
