A positive integer is *magical* if it is divisible by `a` or by `b`. **Return the
`n`-th magical number** (1-indexed), modulo `10^9 + 7`.

**Examples**
```
n = 1, a = 2, b = 3  ->  2
n = 4, a = 2, b = 3  ->  6
n = 5, a = 2, b = 4  ->  10
n = 3, a = 6, b = 4  ->  8
```

**Constraints:** `1 <= n <= 10^9`, `2 <= a, b <= 4*10^4`.
