**Return the number of contiguous sub-arrays of `arr` whose sum is odd**, modulo
`10^9 + 7`.

**Examples**
```
arr = [1,3,5]              ->  4
arr = [2,4,6]              ->  0
arr = [1,2,3,4,5,6,7]      ->  16
arr = [100,100,99,99]      ->  4
```

**Constraints:** `1 <= len(arr) <= 10^5`, `1 <= arr[i] <= 100`.
