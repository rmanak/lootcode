A move increments any single element of `A` by `1`. Return the least number of
moves needed to make every value in `A` distinct.

**Examples**
```
A = [1,2,2]          ->  1     (-> [1,2,3])
A = [3,2,1,2,1,7]    ->  6     (e.g. -> [3,4,1,2,5,7])
```

**Constraints:** `0 <= len(A) <= 4*10^4`, `0 <= A[i] < 4*10^4`.
