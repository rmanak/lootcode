**Return `true` if `arr` (even length) can be reordered so that
`arr[2*i+1] == 2 * arr[2*i]` for every `i`**, else `false`.

**Examples**
```
arr = [3,1,3,6]        ->  false
arr = [4,-2,2,-4]      ->  true
arr = [1,2,4,16,8,4]   ->  false
```

**Constraints:** `0 <= len(arr) <= 3*10^4` (even), `-10^5 <= arr[i] <= 10^5`.
