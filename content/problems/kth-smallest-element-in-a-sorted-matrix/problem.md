Given an `n x n` matrix whose rows and columns are each sorted in ascending order,
return the `k`-th smallest element in the sorted order (counting duplicates, not
distinct values).

**Example**
```
matrix = [[ 1,  5,  9],
          [10, 11, 13],
          [12, 13, 15]], k = 8   ->  13
```

**Constraints:** `1 <= n <= 300`, `1 <= k <= n^2`.
