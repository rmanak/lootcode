**Return the maximum sum of a subset of `nums` that is divisible by `3`** (choose the
empty subset, sum `0`, if nothing better is possible).

**Examples**
```
nums = [3,6,5,1,8]   ->  18   (3 + 6 + 1 + 8)
nums = [4]           ->  0
nums = [1,2,3,4,4]   ->  12
```

**Constraints:** `1 <= len(nums) <= 4*10^4`, `1 <= nums[i] <= 10^4`.
