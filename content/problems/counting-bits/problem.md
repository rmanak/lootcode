Given an integer `n`, return **an array `ans` of length `n + 1`** where `ans[i]`
is the number of set bits (`1`s) in the binary representation of `i`.

## Constraints
- `0 <= n <= 10^5`

## Examples
Input: `n = 2`
Output: `[0,1,1]`

Input: `n = 5`
Output: `[0,1,1,2,1,2]`
Explanation: `3 = 11` has two set bits, `5 = 101` has two.
