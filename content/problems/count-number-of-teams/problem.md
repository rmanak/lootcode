From `n` soldiers with distinct ratings, count triples of indices `i < j < k` whose
ratings are strictly increasing or strictly decreasing. **Return the number of such
teams.**

**Examples**
```
rating = [2,5,3,4,1]  ->  3
rating = [2,1,3]      ->  0
rating = [1,2,3,4]    ->  4
```

**Constraints:** `1 <= n <= 1000`, distinct `1 <= rating[i] <= 10^5`.
