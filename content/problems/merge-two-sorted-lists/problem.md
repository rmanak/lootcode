You are given the heads of two **sorted singly linked lists** `list1` and `list2`.
Merge them into one sorted list by splicing together their nodes, and return the
head of the merged list.

> **Format:** each node is a `ListNode` with a `.val` and a `.next` pointer (the class is provided — do not redefine it). Lists are shown below as the array of their node values (`[]` = empty list).

**Example:** `list1 = [1,2,4]`, `list2 = [1,3,4]` -> `[1,1,2,3,4,4]`.

**Constraints:** `0 <= number of nodes in each list <= 50`; values sorted non-decreasing.
