A *beautiful arrangement* is a permutation `p` of `1..N` such that for every position
`i` (1-indexed) either `p[i]` is divisible by `i` or `i` is divisible by `p[i]`.
**Return how many beautiful arrangements exist.**

**Examples**
```
N = 1  ->  1
N = 2  ->  2
N = 3  ->  3
```

**Constraints:** `1 <= N <= 15`.
