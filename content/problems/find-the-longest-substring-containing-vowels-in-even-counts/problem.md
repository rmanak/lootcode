**Return the length of the longest substring of `s` in which each vowel
(`a, e, i, o, u`) appears an even number of times** (zero counts as even).

**Examples**
```
s = "eleetminicoworoep"  ->  13
s = "leetcodeisgreat"    ->  5
s = "bcbcbc"             ->  6
```

**Constraints:** `1 <= len(s) <= 5*10^5`, lowercase letters.
