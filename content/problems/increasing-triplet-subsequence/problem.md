**Return `true` if there exist indices `i < j < k` with
`nums[i] < nums[j] < nums[k]`.** Aim for `O(n)` time and `O(1)` space.

**Examples**
```
nums = [1,2,3,4,5]  ->  true
nums = [5,4,3,2,1]  ->  false
nums = [2,1,5,0,4,6] ->  true
```

**Constraints:** `1 <= len(nums) <= 5*10^5`, `-2^31 <= nums[i] <= 2^31 - 1`.
