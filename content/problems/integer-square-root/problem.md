Given a non-negative integer `x`, return **`floor(sqrt(x))`** — the largest
integer `r` with `r*r <= x`. Do not use any floating-point square-root function.

## Constraints
- `0 <= x <= 2^31 - 1`.

## Examples
Input: `x = 4`
Output: `2`

Input: `x = 8`
Output: `2`
Explanation: `sqrt(8) ≈ 2.828`, floor is `2`.

Input: `x = 0`
Output: `0`
