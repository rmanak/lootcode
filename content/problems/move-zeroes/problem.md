Given an integer array `nums`, move every `0` to the end while keeping the
relative order of the non-zero elements. Return **the resulting array**.

## Constraints
- `0 <= len(nums) <= 10^5`.
- `-10^9 <= nums[i] <= 10^9`.

## Examples
Input: `nums = [0,1,0,3,12]`
Output: `[1,3,12,0,0]`

Input: `nums = [0]`
Output: `[0]`

Input: `nums = [1,2,3]`
Output: `[1,2,3]`
