A *happy prefix* is a non-empty proper prefix of `s` that is also a suffix.
**Return the longest happy prefix of `s`**, or `""` if none exists.

**Examples**
```
s = "level"         ->  "l"
s = "ababab"        ->  "abab"
s = "leetcodeleet"  ->  "leet"
```

**Constraints:** `1 <= len(s) <= 10^5`, lowercase letters.
