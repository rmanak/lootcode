Starting from `""`, you may repeatedly insert `"abc"` at any position. **Return
`true` if `s` can be produced** this way, else `false`. (`s` consists of `a`, `b`,
`c` only.)

**Examples**
```
s = "aabcbc"        ->  true
s = "abcabcababcc"  ->  true
s = "abccba"        ->  false
```

**Constraints:** `1 <= len(s) <= 2*10^4`, characters are `a`, `b`, `c`.
