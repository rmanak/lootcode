The values of a singly linked list are given in order as an array `head`. Return
**the values after the list is reversed**.

## Constraints
- `0 <= len(head) <= 5000`
- `-5000 <= head[i] <= 5000`

## Examples
Input: `head = [1,2,3,4,5]`
Output: `[5,4,3,2,1]`

Input: `head = [1,2]`
Output: `[2,1]`

Input: `head = []`
Output: `[]`
