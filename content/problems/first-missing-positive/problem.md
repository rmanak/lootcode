Given an unsorted integer array `nums`, return **the smallest positive integer
that does not appear** in it. Aim for `O(n)` time and `O(1)` extra space.

## Constraints
- `0 <= len(nums) <= 10^5`.
- `-10^9 <= nums[i] <= 10^9`.

## Examples
Input: `nums = [1,2,0]`
Output: `3`

Input: `nums = [3,4,-1,1]`
Output: `2`

Input: `nums = [7,8,9,11,12]`
Output: `1`
