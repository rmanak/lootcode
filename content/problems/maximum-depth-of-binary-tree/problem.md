Given the `root` of a binary tree (a `TreeNode` with `value`, `left`, `right`),
return **its maximum depth** — the number of nodes on the longest path from the
root down to a leaf.

Trees are shown below in LeetCode level-order array form (`null` marks a missing
child), but your function receives the `root` as a `TreeNode` (or `None` for an
empty tree).

## Constraints
- `0 <= number of nodes <= 10^4`
- `-100 <= node.value <= 100`

## Examples
Input: `root = [3,9,20,null,null,15,7]`
Output: `3`

Input: `root = [1,null,2]`
Output: `2`
