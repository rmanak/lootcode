Given the level-order array `root` of a binary tree, return **its maximum depth**
— the number of nodes on the longest path from the root down to a leaf.

## Constraints
- `0 <= number of nodes <= 10^4`
- `-100 <= Node.val <= 100`

## Examples
Input: `root = [3,9,20,null,null,15,7]`
Output: `3`

Input: `root = [1,null,2]`
Output: `2`
