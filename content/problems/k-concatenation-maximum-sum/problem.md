Repeat `arr` `k` times to form a longer array. **Return the maximum subarray sum of
that array** (an empty subarray has sum `0`), modulo `10^9 + 7`.

**Examples**
```
arr = [1,2], k = 3    ->  9
arr = [1,-2,1], k = 5  ->  2
arr = [-1,-2], k = 7   ->  0
```

**Constraints:** `1 <= len(arr) <= 10^5`, `1 <= k <= 10^5`, `-10^4 <= arr[i] <= 10^4`.
