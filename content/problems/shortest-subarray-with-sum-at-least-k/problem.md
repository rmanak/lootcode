**Return the length of the shortest non-empty contiguous subarray of `A` with sum at
least `K`**, or `-1` if none exists. `A` may contain negative numbers.

**Examples**
```
A = [1], K = 1        ->  1
A = [1,2], K = 4      ->  -1
A = [2,-1,2], K = 3   ->  3
```

**Constraints:** `1 <= len(A) <= 5*10^4`, `-10^5 <= A[i] <= 10^5`, `1 <= K <= 10^9`.
