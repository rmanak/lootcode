A *132 pattern* is a subsequence `nums[i], nums[j], nums[k]` with `i < j < k` and
`nums[i] < nums[k] < nums[j]`. **Return `true` if such a pattern exists.**

**Examples**
```
nums = [1,2,3,4]    ->  false
nums = [3,1,4,2]    ->  true
nums = [-1,3,2,0]   ->  true
```

**Constraints:** `1 <= len(nums) <= 2*10^4`, integer values.
