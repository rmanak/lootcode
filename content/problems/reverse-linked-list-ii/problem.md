Given the `head` of a **singly linked list** and two integers `m` and `n`, reverse
the nodes from position `m` to position `n` (1-indexed, inclusive) and return the
head of the resulting list.

> **Format:** each node is a `ListNode` with a `.val` and a `.next` pointer (the class is provided — do not redefine it). Lists are shown below as the array of their node values (`[]` = empty list).

**Example**
```
head = [1,2,3,4,5], m = 2, n = 4   ->  [1,4,3,2,5]
```

**Constraints:** `1 <= m <= n <= number of nodes`, `1 <= number of nodes <= 500`.
