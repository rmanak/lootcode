**Return the maximum sum of a non-empty contiguous subarray, with at most one element
deletion allowed** (the subarray must remain non-empty after the deletion).

**Examples**
```
arr = [1,-2,0,3]      ->  4
arr = [1,-2,-2,3]     ->  3
arr = [-1,-1,-1,-1]   ->  -1
```

**Constraints:** `1 <= len(arr) <= 10^5`, `-10^4 <= arr[i] <= 10^4`.
