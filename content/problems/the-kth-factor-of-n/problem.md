A factor of `n` is an integer `i` with `n % i == 0`. List all factors of `n` in
ascending order; **return the `k`-th factor, or `-1` if `n` has fewer than `k`
factors.**

**Examples**
```
n = 12, k = 3  ->  3    (factors 1,2,3,4,6,12)
n = 7, k = 2   ->  7
n = 4, k = 4   ->  -1
```

**Constraints:** `1 <= k <= n <= 1000`.
