Given the `head` of a **singly linked list** and an integer `n`, remove the `n`-th
node from the end of the list and return the head of the resulting list.

> **Format:** each node is a `ListNode` with a `.val` and a `.next` pointer (the class is provided — do not redefine it). Lists are shown below as the array of their node values (`[]` = empty list).

**Example:** `head = [1,2,3,4,5]`, `n = 2` -> `[1,2,3,5]`.

**Constraints:** `1 <= number of nodes <= 30`, `1 <= n <= number of nodes`.
