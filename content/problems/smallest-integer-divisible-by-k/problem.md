Find the smallest positive integer `N` made up only of the digit `1` (a *repunit*:
`1, 11, 111, ...`) that is divisible by `K`. **Return the length of `N`, or `-1` if no
such `N` exists.**

**Examples**
```
K = 1  ->  1    (N = 1)
K = 2  ->  -1
K = 3  ->  3    (N = 111)
```

**Constraints:** `1 <= K <= 10^5`.
