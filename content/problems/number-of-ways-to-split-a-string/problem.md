Split a binary string `s` into three non-empty parts `s1 + s2 + s3 = s`. **Return the
number of splits where `s1`, `s2`, and `s3` contain the same number of `'1'`s**,
modulo `10^9 + 7`.

**Examples**
```
s = "10101"  ->  4
s = "1001"   ->  0
s = "0000"   ->  3
```

**Constraints:** `3 <= len(s) <= 10^5`, `s[i]` is `'0'` or `'1'`.
