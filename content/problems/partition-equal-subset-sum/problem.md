Given an array of positive integers `nums`, return **`true` if it can be split
into two subsets with equal sum**, and `false` otherwise.

## Constraints
- `1 <= len(nums) <= 200`.
- `1 <= nums[i] <= 100`.

## Examples
Input: `nums = [1,5,11,5]`
Output: `true`
Explanation: `[1,5,5]` and `[11]`.

Input: `nums = [1,2,3,5]`
Output: `false`

Input: `nums = [2,2]`
Output: `true`
